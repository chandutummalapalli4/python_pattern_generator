# 🎨 Python Pattern Generator

A menu-driven Python application that generates different star patterns using nested loops and conditional statements. This project was built to practice Python fundamentals and improve logical thinking through pattern programming.

---

## ✨ Features

- Interactive menu-driven interface
- 11 different star patterns
- Input validation
- Runs continuously until the user exits
- Beginner-friendly and well-organized code

---

## 📋 Available Patterns

- Solid Square
- Right Triangle
- Inverted Triangle
- Right-Aligned Triangle
- Pyramid
- Inverted Pyramid
- Diamond
- Hollow Square
- Hollow Right Triangle
- Hollow Pyramid
- Hollow Diamond

---

## 🛠️ Technologies Used

- Python 3
- Visual Studio Code

---

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/chandutummalapalli4/python-pattern-generator.git
```

2. Navigate to the project folder

```bash
cd python-pattern-generator
```

3. Run the program

```bash
python pattern_generator.py
```
