# 🎨 Python Pattern Generator

A menu-driven Python application that generates various star patterns using nested loops and conditional statements. This project was built to strengthen Python fundamentals and improve logical thinking through pattern programming.

---

## ✨ Features

- 📋 Menu-driven interface
- ⭐ 11 different star patterns
- 🔄 Continuous execution until Exit
- ✅ Input validation
- 🧹 Clean and beginner-friendly code

## 🚀 How to Run

### Clone the repository

```bash
git clone https://github.com/chandutummalapalli4/python-pattern-generator.git
```

### Navigate to the project folder

```bash
cd python-pattern-generator
```

### Run the program

```bash
python pattern_generator.py
```
